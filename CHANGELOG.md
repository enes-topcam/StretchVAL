# Changelog

Bu projedeki önemli değişiklikler burada tutulur.

## [1.0.0] - 2026

İlk sürüm.

- Masaüstü çözünürlüğü ve yenileme hızı (Hz) değiştirme, çoklu monitör desteği
- Aspect ratio filtresi (4:3, 5:4 vurgulu) + tek kutuda çözünürlük listesi
- Valorant `GameUserSettings.ini` düzenleyerek true stretched (letterbox off, FullscreenMode=2, çözünürlük)
- Hesap profilleri: kaydet / seç / sil / yeniden adlandır (`named` bayrağı)
- "Devam et" akışı: hesap seç → uygula → Valorant'ı aç
- Yeni hesap sihirbazı: Valorant aç → giriş algıla → kapat → uygula → yeniden aç
- En son seçilen çözünürlüğü hatırlama (otomatik seçili gelir)
